# Contexto do Projeto: Customização Quickshell

Este documento define o contexto para as interações de desenvolvimento e customização da configuração do Quickshell.

## Ambiente
- **Sistema Operacional**: Arch Linux
- **Compositor**: Hyprland
- **Interface Shell**: Quickshell (Qt/QML based)
- **Base**: Configuração inspirada no Omarchy

## Objetivo
Customizar a interface do Quickshell para atender às preferências do usuário, mantendo a compatibilidade com o ambiente Hyprland existente.

## Diretrizes
1. **Tecnologias**: Foco em QML e Javascript (ES6+).
2. **Estilo**: Manter a coerência visual com o tema do Hyprland.
3. **Performance**: Priorizar leveza e responsividade.
4. **Iteração**: Pequenas mudanças incrementais com validação frequente.
